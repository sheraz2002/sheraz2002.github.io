Data Structures & Algorithms Specialization

University of California San Diego · Coursera

Overview: A rigorous 6-course track in algorithmic programming techniques: divide & conquer, greedy, dynamic programming; core data structures (arrays, lists, stacks/queues, heaps, hash tables, BSTs, disjoint sets); graph algorithms (BFS/DFS, topological sort, shortest paths, MST); string algorithms (suffix arrays/trees, KMP, Burrows–Wheeler); and advanced topics (network flows, linear programming, NP-completeness). The capstone focuses on genome assembly concepts with graph models.

Course-by-Course Highlights

  • Algorithmic Toolbox: Sorting/searching, divide & conquer, greedy, dynamic programming; asymptotic analysis and correctness proofs; many programming puzzles.
  • Data Structures: Arrays, stacks/queues, linked lists, hash tables (collision handling), heaps/priority queues, disjoint-set union, binary search trees and balancing ideas.
  • Algorithms on Graphs: BFS/DFS, topological ordering, SCCs, shortest paths (Dijkstra, Bellman-Ford), minimum spanning trees (Kruskal/Prim), bi-directional search.
  • Algorithms on Strings: Pattern matching with KMP, suffix arrays/trees, Burrows–Wheeler transform, FM-index; applications to text and bioinformatics.
  • Advanced Algorithms & Complexity: Max-flow/min-cut, reductions, linear programming, NP-completeness and coping strategies (approximation, randomized methods).
  • Capstone: Genome assembly pipeline concepts (reads → k-mers → de Bruijn graphs → contigs) and practical algorithmic trade-offs.

Why it matters: This specialization sharpened my problem-solving speed and rigor for production code and interviews—choosing the right data structure, proving correctness, and reasoning about time/space trade-offs under constraints.
